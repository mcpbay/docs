# mcpb install-mcp

Installs the MCPB MCP server to an AI tool.

## Syntax

```sh
mcpb install-mcp [options] <target>
```

## Arguments

### options

#### --mcp-name \<name\>

Defines a name for the MCP.

##### name

The name of the MCP server.

### target

#### claudecode

Install the MCPBay MCP Server in Claude Code, user scope.
